# Telegram Parser & Inviter

Инструменты для парсинга участников Telegram-групп и их последующего инвайта в целевые группы.

## Возможности

### Parser (`parser.py`)
- Парсинг участников из групп/каналов
- **Сортировка по времени последней активности** (от самых активных к наименее активным)
- Сохранение результатов в CSV файл
- Отображение топ-N самых активных участников

### Inviter (`inviter.py`)
- Инвайт участников из CSV файла в целевую группу
- Автоматическая обработка ошибок (FloodWait, приватность, уже участник и т.д.)
- Настраиваемая задержка между добавлениями
- Статистика успешных/неуспешных добавлений

## Установка

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройте `.env` файл:**
   
   Скопируйте `.env.example` в `.env` и заполните своими данными:
   ```bash
   # На Windows:
   copy .env.example .env
   
   # На Linux/Mac:
   cp .env.example .env
   ```
   
   Затем отредактируйте `.env`:
   ```
   API_ID=ваш_api_id
   API_HASH=ваш_api_hash
   PHONE=+79991234567
   TARGET_GROUP_ID=-1001234567890
   ```
   
   > Получите API_ID и API_HASH на https://my.telegram.org/apps

## Использование

### 1. Парсинг участников

```bash
python parser.py
```

**Процесс:**
1. Введите username или ID группы для парсинга (например, `@durov` или `-1001234567890`)
2. Укажите максимальное количество участников (по умолчанию 1000)
3. Скрипт отсортирует участников по активности и покажет топ-20
4. Сохраните результаты в CSV файл (по умолчанию `parsed_members.csv`)

### 2. Инвайт участников

```bash
python inviter.py
```

**Процесс:**
1. Укажите путь к CSV файлу с участниками (по умолчанию `parsed_members.csv`)
2. Введите username или ID целевой группы
3. Укажите диапазон задержки между добавлениями (по умолчанию 30-60 секунд)
4. Скрипт начнет добавлять участников с обработкой ошибок

## Структура CSV файла

Файл содержит следующие поля:
- `id` - ID пользователя
- `username` - юзернейм (@username)
- `first_name` - имя
- `last_name` - фамилия
- `phone` - телефон (если доступен)
- `last_activity` - время последней активности

## Важные замечания

⚠️ **Ограничения Telegram:**
- Можно добавлять только пользователей с username
- Некоторые пользователи могут запретить добавление в настройки приватности
- При частых добавлениях может сработать FloodWait (ограничение скорости)
- Рекомендуется делать перерывы между сессиями инвайта

⚠️ **Безопасность:**
- Не используйте свои основные аккаунты для массового инвайта
- Делайте достаточные задержки между добавлениями (30-60 сек минимум)
- Избегайте добавления более 50-80 человек в день с одного аккаунта

## Пример рабочего процесса

```bash
# Шаг 1: Запускаем парсер
python parser.py
# → Вводим: @target_channel
# → Вводим: 500
# → Получаем parsed_members.csv с отсортированными участниками

# Шаг 2: Запускаем инвайтер
python inviter.py
# → Вводим: parsed_members.csv
# → Вводим: @my_group
# → Вводим: 40-60 (задержка)
# → Ждем завершения процесса
```

## Поддержка нескольких аккаунтов

Для использования нескольких аккаунтов создайте отдельные `.env` файлы или модифицируйте скрипты для работы с разными сессиями.
