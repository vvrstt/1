# Telegram Parser & Inviter

Инструмент для парсинга участников Telegram-групп и их инвайта в другие группы через несколько аккаунтов с поддержкой прокси.

## Возможности

✅ **Добавление аккаунтов через интерфейс программы** (не через код)  
✅ **Добавление прокси через интерфейс программы**  
✅ **Парсинг участников с сортировкой по времени последней активности**  
✅ **Инвайт через несколько аккаунтов с распределением нагрузки**  
✅ **Поддержка SOCKS5, HTTP, MTProto прокси**  
✅ **Защита от Flood Wait и других ошибок**  

## Установка

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустите менеджер аккаунтов и прокси
```bash
python manager.py
```

В меню менеджера вы можете:
- Добавить аккаунты Telegram (номер телефона, API ID, API Hash)
- Добавить прокси (тип, IP, порт, логин/пароль)
- Просмотреть список аккаунтов и прокси
- Удалить аккаунты и прокси

### 3. Запустите парсер
```bash
python parser.py
```

Парсер:
- Покажет список доступных аккаунтов
- Позволит выбрать аккаунт для парсинга
- Позволит выбрать прокси (если есть)
- Спарсит участников из указанной группы/канала
- Отсортирует их по времени последней активности (от самых активных к наименее активным)
- Сохранит результат в CSV-файл

### 4. Запустите инвайтер
```bash
python inviter.py
```

Инвайтер:
- Покажет список спаршенных CSV-файлов
- Позволит выбрать файл с пользователями
- Позволит выбрать один или несколько аккаунтов для инвайта
- Добавит пользователей в целевую группу
- Обработает ошибки (FLOOD_WAIT, приватность и т.д.)

## Получение API ключей

1. Зайдите на https://my.telegram.org
2. Авторизуйтесь по номеру телефона
3. Перейдите в "API development tools"
4. Создайте новое приложение
5. Скопируйте **API ID** и **API Hash**

## Структура файлов

- `database.py` - Инициализация базы данных
- `manager.py` - Менеджер аккаунтов и прокси (добавление/удаление/просмотр)
- `parser.py` - Парсер участников с сортировкой по активности
- `inviter.py` - Инвайтер пользователей в группы
- `requirements.txt` - Список зависимостей
- `data.db` - База данных (создается автоматически)

## Важные замечания

⚠️ Не используйте слишком агрессивные настройки инвайта  
⚠️ Делайте перерывы между массовыми действиями  
⚠️ Telegram может ограничить аккаунт за спам  
⚠️ Используйте только качественные прокси  
⚠️ Соблюдайте правила Telegram

## Пример работы

```bash
# Шаг 1: Добавляем аккаунты и прокси
python manager.py

# Шаг 2: Парсим участников
python parser.py

# Шаг 3: Инвайтим в свою группу
python inviter.py
```
