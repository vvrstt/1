# Telegram Parser & Inviter GUI

Полноценное графическое приложение для парсинга участников Telegram и инвайта их в ваши группы.

## 🚀 Установка

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите приложение:**
   ```bash
   python main.py
   ```

## 📋 Возможности

### 1. Вкладка "Аккаунты"
- Добавление аккаунтов Telegram через интерфейс (номер телефона, API ID, API Hash)
- Автоматический запрос кода подтверждения в GUI окне
- Просмотр списка всех добавленных аккаунтов
- Удаление аккаунтов

### 2. Вкладка "Прокси"
- Добавление прокси (SOCKS5, HTTP, MTProto)
- Настройка хоста, порта, логина и пароля
- Просмотр и удаление прокси

### 3. Вкладка "Парсер"
- Выбор аккаунта для парсинга
- Ввод ссылки на группу/канал
- Фильтрация по активности:
  - Все участники
  - Недавно был онлайн
  - Был в течение недели
  - Был в течение месяца
- Сортировка результатов по времени последней активности
- Сохранение в CSV файл

### 4. Вкладка "Инвайтер"
- Выбор CSV файла со спаршенными пользователями
- Выбор аккаунта для инвайта
- Настройка целевой группы
- Настройка задержек между действиями (мин/макс)
- Логирование процесса в реальном времени
- Обработка ошибок (FloodWait, приватность, уже в группе)

## ⚙️ Как получить API ID и API Hash

1. Зайдите на https://my.telegram.org
2. Авторизуйтесь по номеру телефона
3. Перейдите в раздел "API development tools"
4. Создайте новое приложение
5. Скопируйте **API ID** и **API Hash**

## 💡 Советы

- Используйте несколько аккаунтов для распределения нагрузки
- Добавляйте прокси для каждого аккаунта (в текущей версии используется первый доступный прокси)
- Настраивайте задержки между инвайтами (рекомендуется 10-60 секунд)
- Не инвайте больше 50-80 человек в день с одного аккаунта во избежание бана

## 📁 Структура файлов

- `main.py` — основной файл с GUI приложением
- `requirements.txt` — список зависимостей
- `data.db` — база данных (создается автоматически) для хранения аккаунтов и прокси
- `parsed_*.csv` — файлы со спаршенными участниками

## ⚠️ Важное предупреждение

Использование данного инструмента может нарушать условия использования Telegram. Используйте на свой страх и риск. Не рекомендуется использовать на основных аккаунтах.
